# کارتام

ابزاری برای مدیریت و نگه‌داری کارت‌های بانکی (شمارهٔ کارت، شمارهٔ شبا، شمارهٔ حساب و...).

امکانات برنامه:

- پشتیبانی از ۳۷ بانک، نئوبانک و موسسه ایرانی
- ذخیره شماره کارت، شماره شبا، شماره حساب، تاریخ انقضا و CVV2
- کپی، اشتراک‌گذاری و ویرایش کارت

همهٔ اطلاعات در برنامه به‌صورت آفلاین، کدگذاری‌شده و امن در دستگاه شما ذخیره می‌شود.
